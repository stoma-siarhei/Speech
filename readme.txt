На машине надо иметь установленный Python3.x, с настроенными путями к каталогам интерпретатора.
Для компиляции необходимо прописать пути к заголовочным файлам python и библиотеке, если они отличаются от путей прописанных в PATH 
В настройках линковщика необходимо указать версию библиотеки python3x.lib
Скрипт положить в каталог с запускаемой программой
Параметры командной строки 
-script имя запускаемого скрипта
-func имя функции в файле скрипта
-str строка которую необходимо преобразовать в звук
-file имя выходного файлам
-script berk_script -func berk_func -str Hello world -file D:\\test.wav
